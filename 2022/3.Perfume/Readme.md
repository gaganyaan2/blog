# How to make Perfume?

## Basic

Ethnol + Compund + Base + any oil = Perfume
70%    + 20%     + 5%   + 5%
[      15-45 Days       ]

## Bases

1. ISO E Super (woody)
2. Lyral
3. Helional
4. Methyl Ionone
5. Polysantol
6. Galaxolide (musk)
7. Hedione (eg. jasmin flow in air)

C8 - Organge
C16 - Strawberry

C series = syntactic oil
