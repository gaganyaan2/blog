# How to make Perfume?

## Basic

Ethnol + (Compund || Base) + (Perfume essential oil + DPG oil)= Perfume

70%    + 20%     + 5%   + 5%

[      15-45 Days       ]

2. Lyral
3. Helional
4. Methyl Ionone
5. Polysantol
6. Galaxolide (musk)
7. Hedione (floral .eg. jasmin flow in air)

- Type : woody, floral, powdery
